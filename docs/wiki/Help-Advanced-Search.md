## Open advanced search

In Lightroom Classic:

- `Library -> Plug-in Extras -> Advanced search`

## Search scope

- `All photos`
- `Current view`
- `Selected photos`

## Query examples

- `red sports car in front of a garage`
- `sunset over mountains with orange sky`

## How ranking works

The search combines:

- semantic embeddings
- AI-generated metadata fields (`keywords`, `caption`, `title`, `alt_text`)

Results are returned in relevance order and stored in a new Lightroom collection.

## Important Lightroom tip

Set collection sort order to `Custom Order` to see best matches at the top.
