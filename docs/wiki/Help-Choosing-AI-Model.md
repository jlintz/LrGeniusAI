# Help: Choosing an AI Model

> Migrated from `lrgenius.com/help/choosing-ai-model` and curated for repo docs.  
> Dynamic model lists and pricing can change over time.

## Decision factors

Choose based on:

- privacy requirements
- performance/quality expectations
- operating cost
- available hardware

## Cloud models

Typical choices:

- Google Gemini
- OpenAI ChatGPT

Pros:

- best quality with minimal local setup

Trade-offs:

- per-image cost
- images are processed by external services

## Local models

Typical choices:

- Ollama-hosted vision models
- LM Studio-hosted vision models

Pros:

- no per-image API billing
- stronger privacy (local processing)

Trade-offs:

- requires local hardware resources
- setup/maintenance effort

## Practical recommendation

Try at least one cloud model and one local model on your own image set, then compare:

- metadata quality
- runtime per image
- system load
- cost
